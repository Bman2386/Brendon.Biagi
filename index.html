QuestRabbit

QuestRabbit2.0 is the updated version that has been updated from rails 5.2, to rails 7.0.4.2 on the backend. The frontend accordingly needed to be updated from React class componenents (depricated), to modern React functional components (aka React Hooks). The reason for the change was due to modern webhosting removing Rails 5.2 from their stacks.

QuestRabbit is a Full Stack clone of Task Rabbit. In this app you can hire adventurers to complete any and all quests for you. QuestRabbit is also a single page app. It uses JavaScript, CSS, HTML, React, & Redux for the front end. The backend is achieved through Ruby, & Ruby on Rails. PostgreSQL is used for the database.

BUBBLES

Bubbles is an original 2D browser based game where you control a bubble, defeat enemies and if your score gets high enough fight a boss. Bubbles is built with mainly JavaScript and to achieve complete functionality, HTML, CSS, and Canvas. It is recommended that you play with the sound on as there are actual sound elements built into the game ;)

Take A Hike

Take A Hike is an app for finding trails to hike on developed with the MERN stack. It uses MongoDB for the database, with JavaScript both in the backend and frontend. The front end componenents use google API to find actual trails you can hike on and leave reviews about. The front end also uses React, Redux, HTML, and CSS, and is a single page app.
