Take A Hike

Take A Hike is an app for finding trails to hike on developed with the MERN stack. It uses MongoDB for the database, with JavaScript both in the backend and frontend. The front end componenents use google API to find actual trails you can hike on and leave reviews about. The front end also uses React, Redux, HTML, and CSS, and is a single page app.
